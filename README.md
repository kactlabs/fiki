# Fiki - Food Wiki

An AI-powered food recipe wiki generator that creates detailed, authentic recipes from cuisines around the world.

## Features

- 🍳 Generate random international recipes using LLM
- 📝 Automatic markdown file creation with consistent formatting
- 🔢 Sequential numbering system (001, 002, 003...)
- 🚫 Duplicate detection to avoid creating the same recipe twice
- 📚 Automatic index.md generation with all recipe links
- 🌍 ASCII-safe filenames (removes special characters)
- 🤖 Tracks which LLM model generated each recipe
- ⏰ Timestamps for each generated recipe

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Configure your LLM provider in `.env`:
```bash
cp .env.sample .env
# Edit .env with your settings
```

## Usage

### Generate Recipes

Create multiple random recipes:
```bash
python business.py 3
```

This will:
- Generate 3 unique recipes
- Save them as `001-dish-name.md`, `002-dish-name.md`, etc.
- Track dish names in `unique_dishes.txt` to prevent duplicates
- Automatically update `index.md` with links to all recipes
- Add metadata (LLM model and timestamp) to each recipe

### Update Index

Manually regenerate the index page:
```bash
python indexer.py
```

### Fix Special Characters

Remove accents and special characters from existing filenames:
```bash
python character_fix.py
```

Converts: `005-bánh-xèo.md` → `005-banh-xeo.md`

## File Structure

```
.
├── business.py              # Main recipe generator
├── indexer.py              # Index page generator
├── character_fix.py        # Filename sanitizer
├── llm.py                  # LLM adapter (supports multiple providers)
├── index.md                # Main index page with all recipes
├── unique_dishes.txt       # Tracks generated dishes
├── sample-template.md      # Recipe template format
├── 001-dish-name.md        # Generated recipe files
├── 002-dish-name.md
└── ...
```

## Recipe Format

Each recipe follows a consistent markdown format:

```markdown
/ [Home](index.md)

# Dish Name (Origin)

Brief description of the dish.

---

## Ingredients
...

## Recipe Steps
...

---

*Generated by: ollama-llama3.2:3b*  
*Generated time: 2026-03-09 14:30:45*
```

## LLM Support

Supports multiple LLM providers via `llm.py`:
- Ollama
- OpenAI
- Google Gemini
- llama.cpp

Configure your provider in `.env` file.
